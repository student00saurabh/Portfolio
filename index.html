Major Projects

These are my top 5 major projects that demonstrate my ability to build real-world, scalable and responsive applications using modern web technologies.

Wanderlust

Wanderlust is a full-stack room listing web application where users can explore, post, and book stays. It features authentication, review system, cloud image hosting, and search filters. Built using Node.js, Express, MongoDB (Atlas), EJS, Passport.js, Cloudinary, Bootstrap, HTML/CSS/JS and deployed on Render.

FriendZone

FriendZone is a full-featured social media web application where users can sign up, create posts, like, comment, and send private messages. Built using Node.js, Express, MongoDB, EJS, Passport.js, Socket.IO and other backend libraries. The platform is responsive, secured, and includes real-time chat integration.

Chat Room

Chat Room is a real-time full-stack messaging app allowing users to log in, join chat rooms, and communicate instantly. Built using Node.js, Express, MongoDB, EJS, Socket.IO, and Passport.js. The app handles multiple users simultaneously and includes proper user sessions and real-time message broadcasting.

AlphaMovies

AlphaMovies is a modern React-based movie search and explore app powered by OMDB API. Users can search by title, view detailed info, and save favorites. It features **pagination, category filtering, and responsive card layouts** using React, Axios, CSS, and Vercel for deployment.
